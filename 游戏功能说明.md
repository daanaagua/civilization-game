# 文明发展游戏 - 完整功能说明

## 游戏概述

这是一款基于Next.js 14开发的文明建设类放置游戏，玩家从原始部落开始，通过资源管理、建筑建造、科技研发、人物培养等方式，逐步发展自己的文明。游戏采用现代化的Web技术栈，提供流畅的游戏体验和精美的用户界面。

## 核心游戏系统

### 1. 资源管理系统

#### 资源概览
- **食物** 🌾：维持人口生存的基础资源，可通过点击手动收集
- **木材** 🪵：建筑和工具制作的重要材料，可通过点击手动收集
- **石料** 🪨：建筑和工具制作的重要材料，可通过点击手动收集
- **其他资源** ：参考resources.md

#### 资源交互特性
- **悬浮提示**：鼠标悬停在资源上时显示详细信息，包括：
  - 当前数量和存储上限
  - 变化速率（+X.X/秒）
  - 资源描述和用途
  - 特殊属性（如人口对应的住房容量）
- **手动收集**：基础资源（食物、木材、石料）支持点击收集
- **自动生产**：通过建筑和工人分配实现自动资源生产
- **存储限制**：每种资源都有存储上限，可通过建筑扩展

### 8. 成就系统（详见achievements.md）

#### 成就分类
- **发展成就**：基础建设和资源发展里程碑
- **科技成就**：科技研发相关成就
- **军事成就**：军事发展相关成就
- **文化成就**：文化和社会发展成就
- **特殊成就**：隐藏和挑战性成就

#### 成就奖励
- **永久加成**：每个成就提供1%的相关领域加成
- **继承点数**：成就提供继承点用于下一周目
- **解锁内容**：某些成就解锁新功能或建筑

### 9. Buff系统

#### Buff来源
- **继承点购买**：使用继承点购买永久加成
- **事件奖励**：某些事件提供临时或永久Buff
- **人物特性**：人物为文明提供持续加成
- **建筑效果**：建筑提供各种加成效果
- **科技效果**：科技研发带来的永久提升

#### Buff效果
- **资源生产**：提高各种资源的生产效率
- **建筑效率**：提高建筑的工作效率
- **研究速度**：加快科技研发速度
- **稳定度**：提供稳定度加成
- **军事力量**：增强军事能力
- **其他**：待补充

### 10. 继承系统（转生系统）

#### 转生机制
- **继承点计算**：基于游戏进度计算继承点奖励
- **文明重置**：转生后重新开始，但保留继承点
- **永久提升**：使用继承点购买永久加成
- **多周目累积**：继承点和加成在多周目间累积

#### 继承商店
- **资源加成**：提高各种资源的生产效率
- **建筑加成**：降低建筑成本或提高效率
- **研究加成**：提高科技研发速度
- **稳定度加成**：提供基础稳定度提升
- **存储扩容**：增加资源存储上限

## 技术架构

### 前端技术栈
- **Next.js 14**：现代React框架，支持App Router
- **TypeScript**：类型安全的JavaScript超集
- **Tailwind CSS**：实用优先的CSS框架
- **Zustand**：轻量级状态管理库
- **Lucide React**：现代化图标库

### 核心特性
- **服务端渲染**：更好的性能和SEO
- **类型安全**：完整的TypeScript类型定义
- **状态持久化**：游戏状态自动保存到localStorage
- **模块化设计**：清晰的代码结构和组件分离
- **性能优化**：合理的组件拆分和渲染优化

### 项目结构
```
src/
├── app/                 # Next.js App Router
├── components/          # React组件
│   ├── ui/             # 基础UI组件
│   ├── layout/         # 布局组件

```

## 游戏玩法指南

### 新手入门
1. **开始游戏**：点击"开始"按钮启动游戏
2. **收集资源**：点击食物、木材、石料图标手动收集
3. **建造建筑**：使用资源建造茅草屋增加住房
4. **研发科技**：研发基础科技解锁新功能
5. **管理人口**：平衡人口增长和资源消耗

### 进阶策略
1. **资源平衡**：合理分配资源，避免短缺
2. **科技规划**：制定科技发展路线
3. **建筑布局**：优化建筑配置提高效率
4. **人物培养**：招募合适的人物提供加成
5. **军事发展**：建立军队保护文明安全

### 高级技巧
1. **事件应对**：根据情况选择最优事件选项
2. **稳定度管理**：维持高稳定度促进发展
3. **继承规划**：合理使用继承点获得优势
4. **成就追求**：完成成就获得永久加成
5. **多周目发展**：利用继承系统实现更快发展
